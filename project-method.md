# Méthode projet

## Pour ajouter un nouveau projet
1. créer le nouveau repo
2. ajouter un README minimal
3. relier le projet au centre dans le synopsis
4. créer une Issue de consigne dans le centre
5. envoyer la tâche à un robot
6. faire valider la tâche
7. mettre à jour la progression
8. mettre à jour la reprise si nécessaire

## Ordre conseillé
- d’abord le centre
- ensuite le repo projet
- ensuite la tâche
- ensuite l’exécution
- ensuite la validation
- ensuite la sauvegarde
