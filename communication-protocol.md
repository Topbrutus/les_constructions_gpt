# Communication Protocol

## Démarrage déun robot
1. lire les consignes
2. lire le synopsis
3. vérifier la tâche en cours
4. agir
5. laisser une trace

## Règles de passage
- une seule tâche à la fois
- toujours laisser une trace claire
- reprendre exactement où le travail s’est arrêté

## Statut d’une tâche

- Terminée :
  la tâche demandée est exécutée et confirmée

- Transférée :
  la tâche doit être reprise par un autre robot

- Bloquée :
  la tâche ne peut pas avancer et la raison est écrite clairement

- À reprendre :
  la tâche est commencée mais pas terminée, avec le prochain point exact
